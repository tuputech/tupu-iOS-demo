## Demo

clone整个仓库之后，进入到Example，执行

```
pod install

```

或者


```
pod install --no-repo-update (推荐) 

```

## 安装

TupuTech is available through [CocoaPods](http://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod "TupuTech"
```
